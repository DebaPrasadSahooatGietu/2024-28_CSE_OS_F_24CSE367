##  Description

These assignments cover fundamental and advanced concepts of **Operating Systems**, including:

- Process Management  
- CPU Scheduling Algorithms  
- Memory Management  
- Page Replacement Algorithms  
- Deadlocks  
- File Systems  
- Disk Scheduling  using FCFS (FIRST COME FIRST SERVE)

Each assignment includes theoretical explanations, problem-solving, and in some cases, algorithm implementations.

---

##  Objective

The purpose of this collection is to:

- Strengthen understanding of OS concepts  
- Provide structured academic solutions  
- Serve as a reference for exam preparation  
- Help in practical implementation of algorithms
